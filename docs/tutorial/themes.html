ZCrud tutorial - Themes

A theme in ZCrud is simply a folder in themes directory containing a CSS file and some images (optional).

Nowadays only are 2 themes available:

  • basic theme. Green colors
  • soft theme. Blue colors

For customizing existing themes and building new ones, go to Themes reference page.

To define the theme to use you must to declare it in your web pages! 

<link rel="stylesheet" type="text/css" href="/zcrud/themes/[your-theme-name]/theme.css" media="all">

For example, to use the soft theme yo must declare this: 

<link rel="stylesheet" type="text/css" href="/zcrud/themes/soft/theme.css" media="all">
