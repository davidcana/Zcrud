ZCrud tutorial - Themes

A theme in ZCrud is simply a folder in themes directory containing a CSS file and some images (optional).

Nowadays only is 1 theme available: basic theme.

If you don't like it you can:

  • Customize basic theme. Customize the CSS file and/or the images.
  • Create a new theme. Use the basic theme as a base and customize it. To do this simply copy the basic directory in the themes directory and give it another name.

Important: Be careful with the changes! Don't break the general structure of the CSS file! If you do this you can break ZCrud!

If you create a new theme don't forget to declare it in your web pages! You must replace this: 

<link rel="stylesheet" type="text/css" href="/zcrud/themes/basic/basic.css" media="all">

...by this: 

<link rel="stylesheet" type="text/css" href="/zcrud/themes/your-theme-name/your-theme-name.css" media="all">
