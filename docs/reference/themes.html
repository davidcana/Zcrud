ZCrud API reference - Themes

Built-in themes

A theme in ZCrud is simply a folder in themes directory containing a CSS file and some images (optional). Built-in themes uses Sass, an extension of CSS to make it easy to use variables, nested rules, mixins, inline imports, and more, all with a fully CSS-compatible syntax.

Basic theme is a theme built from scratch. It contains the next files:

  • images. A directory with images.
  • _base.scss. It contains definitions of configuration vars.
  • _colors-physical.scss. Definitions of vars about colors. It defines some names using colors.
  • _colors-logical.scss. Definitions of vars about colors. It assigns colors using the names defined in _color-physical.scss.
  • _common.scss. The core of the styles.
  • _fonts.scss. Definitions of vars about fonts and web fonts.
  • theme.css. The resulting CSS file.
  • theme.css.map. The source map of the CSS file. Developer tools such as those in Firefox and Chrome will use them to allow debugging.
  • theme.scss. The main Sass file for this theme.

Soft theme is a theme built over Basic theme. It contains the next files:

  • _base.scss (not required, but usually exists).
  • _colors-physical.scss (not required, but usually exists).
  • theme.css (required). A theme can not work without this file!
  • theme.css.map (not required but recommended).
  • theme.scss (required if the theme is based on Sass).

Extending themes

If you don't like built-in themes you can:

  • Customize an existing theme.
  • Create a new theme.

Customize an existing theme

Customize the SCSS files and/or the images. If you customize the SCSS files you must rebuild the CSS file running this: 

grunt sass

Create a new theme

Use the basic or soft theme as a base. Simply copy the directory of the base theme in the themes directory and give it another name. Then customize it. You must also add to the files in sass target of Gruntfile.js file. The default files are: 

grunt.initConfig({
    ...
    sass: {
        dist: {
            files: {
                'themes/basic/theme.css': 'themes/basic/theme.scss',
                'themes/soft/theme.css': 'themes/soft/theme.scss'
            }
        }
    }
    ...
});

To add an new theme called hard the files must be: 

grunt.initConfig({
    ...
    sass: {
        dist: {
            files: {
                'themes/basic/theme.css': 'themes/basic/theme.scss',
                'themes/soft/theme.css': 'themes/soft/theme.scss',
                'themes/hard/theme.css': 'themes/hard/theme.scss'
            }
        }
    }
    ...
});

Important: Be careful with the changes! Don't break the general structure of the CSS file! If you do this you can break ZCrud!
