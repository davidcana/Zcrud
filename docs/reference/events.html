ZCrud API reference - Events

User's actions can trigger some events:

  • formClosed( data, event ) -
    Triggered when a form dialog is closed. data argument has some fields:
    • data.$form. The form as jQuery selection.
    • data.formType. Possible values are create, edit, delete or list.
    • data.options. The full configuration options object.
  • formCreated( data ) -
    Triggered when a form dialog is created. data argument has some fields:
    • data.$form. The form as jQuery selection.
    • data.formType. Possible values are create, edit, delete or list.
    • data.options. The full configuration options object.
  • formSubmitting( data, event ) boolean
    Triggered when a form is submitted. data argument has some fields:
    • data.$form. The form as jQuery selection.
    • data.formType. Possible values are create, edit, delete or list.
    • data.dataToSend. The created/updated/deleted records.
    • data.options. The full configuration options object.
    Form validation can be done using this event. If it returns false, submit operation is cancelled.
  • recordAdded( data, event ) -
    Triggered when a new record is added on server side. data argument has some fields:
    • data.record. The added record.
    • data.serverResponse. The response JSON object returned from server.
    • data.options. The full configuration options object.
  • recordDeleted( data, event ) -
    Triggered when an existing record is deleted on server side. data argument has some fields:
    • data.record. The deleted record.
    • data.serverResponse. The response JSON object returned from server.
    • data.options. The full configuration options object.
  • recordUpdated( data, event ) -
    Triggered when an existing record is updated on server side. data argument has some fields:
    • data.record. The updated record.
    • data.serverResponse. The response JSON object returned from server.
    • data.options. The full configuration options object.
  • selectionChanged( data, event ) -
    Triggered when selected rows on the table change. data argument has some fields:
    • data.$rows. The current selection as a jQuery object.
    • data.records. The selected records.
    • data.options. The full configuration options object.
