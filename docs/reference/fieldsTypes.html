ZCrud API reference - Fields - Field types

Field types

The list of field types is:

  • checkbox boolean none
    Useful to manage true/false values.

  • checkboxes array of strings addCurrentValueToOptions, dependsOn, options, translateOptions
    Useful to allow to select multiple options. Checkboxes is a suitable choice when you want to present a lot of options to the user and give them the chance to choose one or more items.

  • datalist string addCurrentValueToOptions, dependsOn, options, translateOptions
    An input element with pre-defined values.

  • date date inline, minYear, maxYear
    Manages date values (no time part).

  • datetime date inline, minYear, maxYear, maxHour, minutesStep, timerDelay
    Manages date and time values.

  • radio string addCurrentValueToOptions, dependsOn, options, translateOptions
    Radio buttons. Makes it easy for selecting one of many choices.

  • select string addCurrentValueToOptions, dependsOn, options, translateOptions
    It is used to create a drop-down list. Similar to radio buttons (for selecting one of many choices).

  • subform object buttons, fields, subformKey
    Defines a form inside another form. Makes it easy to manage master/child tables.

  • text string none
    Defines a one-line text input field.

  • textarea string none
    Defines a multi-line text input field.

  • time string inline, maxHour, minutesStep, timerDelay
    Manages time values (no date part).
