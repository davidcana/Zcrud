ZCrud API reference - Buttons - Button types

Buttons

The buttons in pages and the buttons in fields defines lists of HTML buttons. An example: 

buttons: {
    toolbar: [ 'undo', 'redo', 'form_cancel', 'form_submit' ],
    byRow: [ 'list_deleteRow' ]
}

The toolbar element defines the list of buttons shown in the toolbar (a list of buttons usually shown in the bottom area). The byRow element defines the list of buttons shown in lists and subforms at every row.

Binds

A type of button can be bindable to a specific target, to several targets ot to any target. The available targets are:

  • formToolbar: the toolbar of a form, usually at the bottom of the form.
  • listToolbar: the toolbar of a list, usually at the bottom of the list.
  • listRow: a group of buttons repeated for every row of a list.
  • subformToolbar: the toolbar of a subform, usually at the bottom of the subform.
  • subformRow: a group of buttons repeated for every row of a subform.

Button types

The list of button types is:

  • undo formToolbar and listToolbar
    Undo last user's action.
  • redo formToolbar and listToolbar
    Redo an user's action.
  • generic any
    Generic button. Must be customized.
  • form_cancel formToolbar
    Cancel all non saved changes and return to previous list or form.
  • form_submit formToolbar
    Submit data to server and return to previous list or form (if any).
  • form_copySubformRows formToolbar
    Copy rows from a subform to another. This button can be configured with some custom properties:
    • source (String). The id of the subform working as source.
    • target (String). The id of the subform working as target.
    • onlySelected (boolean). Use all or only the selected items.
    • removeFromSource (boolean). Remove selected items from source after copying them.
    • deselect (boolean). Deselect items from source after copying them.
  • list_showCreateForm listToolbar
    Show a form to create a new record.
  • list_submit listToolbar
    Submit data to server.
  • list_addNewRow listToolbar
    Add a new empty row to the list page.
  • list_showEditForm listRow
    Show a form to edit an existing record.
  • list_showDeleteForm listRow
    Show a form to delete an existing record.
  • list_deleteRow listRow
    Delete the current row of the list page.
  • subform_addNewRow subformToolbar
    Add a new empty row to the subform.
  • subform_showCreateForm subformToolbar
    Show a form to create a new record.
  • subform_showDeleteForm subformRow
    Show a form to delete an existing record.
  • subform_deleteRow subformRow
    Delete the current row of the subform.
  • subform_showEditForm subformRow
    Show a form to edit an existing record.
