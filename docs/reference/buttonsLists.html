ZCrud API reference - Buttons - Lists of buttons

We have 3 ways to define a list of fields:

  1. Refer to a button using its type. An example: 
    buttons: {
    toolbar: [ 'undo', 'redo', 'form_cancel', 'form_submit' ],
    byRow: [ 'list_deleteRow' ]
}
  2. Refer to a button using its type but customizing it. An example: 
    buttons: {
    toolbar: [ 
        'undo', 
        'redo', 
        'form_cancel',
        {
            type: 'form_submit',
            textsBundle: {
                title: undefined,
                content: {
                    translate: false,
                    text: 'Send now!'
                }  
            }
        }
    ],
}
  3. Define a full button instance. An example: 
    buttons: {
    toolbar: [ 
        'undo', 
        'redo', 
        'form_cancel', 
        'form_submit', 
        {
            type: 'generic',
            textsBundle: {
                title: undefined,
                content: {
                    translate: false,
                    text: 'Add members'
                }  
            },
            run: function( event ){
                // Your code here
            }
        }
    ]
}
