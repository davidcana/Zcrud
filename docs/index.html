What is zCrud

zCrud is a jQuery plugin that is used to create AJAX based CRUD tables and forms without coding HTML or Javascript. It has several features including:

  • Automatically creates HTML table and loads records from server using AJAX.
  • Creates create new record forms. When user creates a record, it sends data to server using AJAX and adds the same record to the table in the page.
  • Creates edit record forms. When user edits a record, it updates server using AJAX and updates all cells on the table in the page.
  • Allow user to delete record. When user deletes a record, it deletes the record from server using AJAX and delete the record from the table in the page.
  • All tables and forms are created using templates. It is also possible to use customized templates. The template engine is ZPT-JS.
  • Makes it easy to customize already implemented form field types; also to add new form field types.
  • All styling of table and forms are defined in a CSS file, so you can easily change style of everything. CSS file is well defined and commented.
  • It comes with pre-defined color themes.
  • Supports server side paging, sorting and filtering using AJAX.
  • Supports master/child tables in forms.
  • Allows user to select rows.
  • It is not depended on any server side technology.
  • It uses jquery form validator that makes it easy to validate user input. Exposes some events to enable custom validation.
  • Built-in support for english and spanish (full I18n and L10n support). Other languages can be added easily.
  • All HTML code is valid (w3c compliant). It works on all common browsers.
  • Editable lists support.
  • Undo/redo support.
